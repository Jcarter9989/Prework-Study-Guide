HTML

  • The head element contains information about webpage.
  • The body element represents the visble content shown to the user
  • HTML can include writing, links, pictures, and even sound and video
  • HTML website developers use to create there website

CSS

  • A margin indicates how much we want around the outside of a element
  • A padding indicates how much space we want around the content inside an element
  • Css describes how elements should be rendered on screen or other media
  • Css used for describing the presentation of document written in a mark up language such as HTML

Git

  • Git is a DevOps tool used for source code management
  • Git tracks changes in source code and lets multiple developers to work together on nonlinear development
  • Git status: checks what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switch to it

Javascript

  • JS handles logic and allow users to interact with website
  • JS computer programming language commonly used to create interactive effects within web browsers
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
